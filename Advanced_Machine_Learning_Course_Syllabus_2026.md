# Machine Learning: From Foundations to Production + Generative AI

**Duration:** 14 Weeks (56 Hours)\
**Level:** Intermediate (College Students / Working Professionals)

------------------------------------------------------------------------

# Course Objectives

Students will learn to:

-   Build end-to-end ML systems
-   Understand mathematical foundations
-   Develop production-ready ML pipelines
-   Deploy ML applications
-   Build Deep Learning models
-   Build LLM and RAG applications
-   Apply MLOps best practices
