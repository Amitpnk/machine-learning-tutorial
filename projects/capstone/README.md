# Capstone Project

**Due: End of Week 14**

## Objective

Design, build, and deploy a production-ready ML or Generative AI system that demonstrates every skill from the course.

## Requirements

### 1. Problem & Data (Week 11)
- [ ] Clear problem statement with business impact framing
- [ ] Dataset sourcing with documentation of provenance and license
- [ ] EDA notebook with actionable insights

### 2. Modeling (Week 12)
- [ ] Baseline model + at least 2 improved models
- [ ] Hyperparameter optimization (Optuna recommended)
- [ ] Evaluation on a proper held-out test set

### 3. Explainability (Week 13)
- [ ] SHAP global feature importance plot
- [ ] LIME local explanation for at least 3 individual predictions

### 4. MLOps (Week 13)
- [ ] MLflow experiment tracking (all runs logged)
- [ ] DVC for dataset versioning
- [ ] GitHub Actions CI: runs tests on every push to main

### 5. Deployment (Week 14)
- [ ] FastAPI REST endpoint with `/predict` route
- [ ] Dockerfile + `docker-compose.yml`
- [ ] Deployed to AWS SageMaker, Azure ML, or Hugging Face Spaces (at least one)
- [ ] Basic data drift monitoring configured

### 6. Documentation & Presentation
- [ ] GitHub repository with clean README
- [ ] Architecture diagram (system design)
- [ ] 10-minute demo presentation with slides
