# Module 14 — Capstone Project

**Week 14 · End-to-End Production ML System**

## Overview

Students design, build, and deploy a complete ML or GenAI system that demonstrates every skill from the course.

## Requirements

Your project must include all of the following:

- [ ] Problem framing and dataset selection (with justification)
- [ ] Data collection, cleaning, and feature engineering
- [ ] Model selection, training, evaluation, and hyperparameter tuning
- [ ] Model explainability (SHAP or LIME)
- [ ] MLOps pipeline: MLflow experiment tracking + DVC data versioning
- [ ] CI/CD: GitHub Actions workflow for automated testing
- [ ] Deployment: REST API (FastAPI) + cloud endpoint or Docker container
- [ ] Monitoring: data drift detection setup

## Project Ideas

| Domain | Project |
|--------|---------|
| Finance | Real-time Fraud Detection System |
