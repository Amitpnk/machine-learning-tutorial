# Module 10 — NLP & Generative AI

**Week 10 · 4 Hours**

## Learning Objectives

- Preprocess text: tokenization, stemming, lemmatization, and embeddings
- Understand the Transformer architecture: self-attention, multi-head attention, positional encoding
- Fine-tune BERT for classification tasks using HuggingFace
- Engineer effective prompts: zero-shot, few-shot, chain-of-thought
- Build a RAG pipeline using embeddings, a vector database, and an LLM
- Design a LangChain agent with tool use
