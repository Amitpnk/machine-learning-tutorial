# Module 04 — Supervised Learning: Regression

**Week 4 · 4 Hours**

## Learning Objectives

- Fit Linear Regression using Gradient Descent and the Normal Equation
- Apply Polynomial Regression to capture non-linear patterns
- Understand L1 (Lasso) and L2 (Ridge) regularization and their effects on coefficients
- Combine both with ElasticNet and tune the mixing ratio
- Evaluate models with MAE, RMSE, and R²
