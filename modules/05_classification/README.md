# Module 05 — Supervised Learning: Classification

**Week 5 · 4 Hours**

## Learning Objectives

- Apply Logistic Regression and interpret the decision boundary
- Build Decision Trees and understand Gini impurity vs Entropy
- Use Random Forest and extract feature importances
- Train SVMs with different kernels and understand the margin concept
- Handle imbalanced datasets with SMOTE, class weights, and resampling
- Evaluate classifiers with confusion matrix, ROC curve, and AUC
