# Module 07 — Unsupervised Learning

**Week 7 · 4 Hours · ⭐ Mid-term Project Due**

## Learning Objectives

- Apply K-Means clustering and choose k using the Elbow Method and Silhouette Score
- Build hierarchical clustering and interpret dendrograms
- Use DBSCAN for noise-robust, parameter-free clustering
- Reduce dimensionality with PCA, t-SNE, and UMAP
- Detect anomalies with Isolation Forest and One-Class SVM
