# Module 09 — Computer Vision

**Week 9 · 4 Hours**

## Learning Objectives

- Understand convolution, feature maps, and receptive fields
- Navigate CNN architectures from LeNet to EfficientNet and ResNet
- Apply transfer learning by freezing pretrained layers and adding custom heads
- Use data augmentation to reduce overfitting on small image datasets
- Run real-time object detection with a pretrained YOLO model
