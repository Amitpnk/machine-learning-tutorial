# Module 01 — Python for Machine Learning

**Week 1 · 4 Hours**

## Learning Objectives

- Understand the ML landscape: AI vs ML vs Deep Learning
- Write Pythonic code using OOP, list comprehensions, and generators
- Manipulate arrays efficiently with NumPy (broadcasting, vectorized ops)
- Wrangle tabular data with Pandas (groupby, merge, time series)
- Visualize data with Matplotlib and Seaborn
- Use the Scikit-learn API pattern: `fit → transform → predict`
