# Module 11 — Time Series & Recommender Systems

**Week 11 · 4 Hours**

## Learning Objectives

- Decompose time series into trend, seasonality, and noise
- Fit ARIMA/SARIMA models and diagnose stationarity
- Use Prophet for business forecasting with holidays and seasonality
- Build an LSTM-based time series forecaster
- Implement collaborative filtering for recommendations
- Apply Matrix Factorization (SVD/ALS) for latent factor models
