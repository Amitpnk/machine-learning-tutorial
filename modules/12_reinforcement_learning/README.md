# Module 12 — Reinforcement Learning

**Week 12 · 4 Hours**

## Learning Objectives

- Model problems as MDPs: define states, actions, rewards, and transitions
- Solve the Bellman equation and understand value functions
- Implement tabular Q-Learning on a gridworld environment
- Train a Deep Q-Network (DQN) using PyTorch
- Understand Policy Gradient methods and the REINFORCE algorithm
