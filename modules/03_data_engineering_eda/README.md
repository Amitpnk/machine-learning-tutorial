# Module 03 — Data Engineering & Exploratory Data Analysis

**Week 3 · 4 Hours**

## Learning Objectives

- Load data from CSV, Excel, JSON, SQL, and REST APIs
- Handle missing values, outliers, duplicates, and encoding issues
- Apply feature scaling and categorical encoding correctly
- Perform univariate and bivariate analysis with visualizations
- Build a reusable Scikit-learn preprocessing pipeline
- Understand and prevent data leakage
